Logic Explanation

Source of Truth: I use my array as the single source of truth. The interface does not store the data; it only reflects what is currently saved in the array.

Data Model: I added a quantity property to my plain object literal. This allows the system to group multiples of the same item, and my loop calculates the total by multiplying the value by the quantity.

Reactive Rendering: I created a central updateUI() function that handles all calculations and rendering. Every time the array is modified or the party size changes, updateUI() clears the old display and renders the current state.

State & Logic: I use conditional logic to protect state integrity by validating inputs before they enter the array. I also use it to enable or disable the Split Loot button based on whether the current state is valid.
