Phase 3 Reflection

Persistence Lifecycle: Data is grouped into an object, turned into a text string, and saved to the browser. On page load, the string is loaded, turned back into an object, and checked for errors before being used.

The Role of updateUI(): The updateUI() function must run after loading saved data to make the screen match the background data.

JSON Concepts: JSON.stringify() turns JavaScript objects into text. JSON.parse() turns text back into JavaScript objects.

Debugging Observation: When setting a pause point inside the restore function, the loaded data paused in memory. The data was inspected to make sure the values were valid before the screen updated.

Course Outcomes: This phase supports the goal of creating applications that save user input by keeping data after a page refresh and adding a reset button.
